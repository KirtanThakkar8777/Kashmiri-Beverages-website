# 🏔️ Kashmiri Beverages – Landing Page

This is my **first full-stack web development project**, a landing page inspired by traditional Kashmiri beverages.  
Along with frontend design, this project also includes basic backend features like authentication and storing contact messages.

---

## 🚀 About the Project

The website showcases popular Kashmiri drinks like **Jeera** and **Orange** with a clean and modern UI.

In addition to the landing page, I have implemented:
- User authentication (basic)
- Contact form with database storage

This project helped me understand how frontend and backend work together.

---

## ✨ Features

- 🍵 Landing page showcasing Kashmiri beverages  
- 🔐 Basic user authentication  
- 📩 Contact form (messages stored in database)  
- 📱 Responsive design  
- ⚡ Smooth and simple UI  

---

## 🧠 What I Learned

- Connecting frontend with backend (API integration)  
- Creating and managing REST APIs  
- Handling user authentication  
- Working with databases  
- Structuring a full-stack project
